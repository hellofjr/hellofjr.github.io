关于服务器搭建

#VPS搭建ShadowSocks翻墙
学习网址：https://www.diycode.cc/topics/738

VPS提供商–Vultr.com

  • 注册账号
    • http://www.vultr.com/?ref=7038906注册
    • 可使用支付宝充值，服务器按天计费
  • 创建Server
    • 在Vultr Server页面，点击右上角“+”按钮
    • Los Angelos节点我在用，感觉还可以，看youtube视频没有压力
    • 操作系统CenterOS，选择的是5美元/月，2.5美元/月的sold out
    • 创建成功后，Servers界面会有显示，等待状态变成Running
  • 连接服务器
    • 点击刚才创建的Server进入详情页面
    • 页面显示该VPS的IP、Username、Password
  • 远程登录VPS
    • mac用户直接ssh登录ssh Username@IP
    • win用户推荐使用putty软件
  • 修改登录密码
    • 输入passwd,要求输入新密码，按提示操作

搭建ShadowSocks

VPS安装ShadowSocks

  • 推荐看秋水逸冰的《Shadowsocks一键安装脚本（四合一）》的安装教程。

  • 运行以下命令

    wget --no-check-certificate -O shadowsocks-all.sh https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-all.sh
    chmod +x shadowsocks-all.sh
    ./shadowsocks-all.sh 2>&1 | tee shadowsocks-all.log

  • 选择脚本（Python、R、Go、libev），任选一个

    Which Shadowsocks server you’d select:
    1.Shadowsocks-Python
    2.ShadowsocksR
    3.Shadowsocks-Go
    4.Shadowsocks-libev
    Please enter a number (default 1):

    选择ShadowSocks-Go，输入3…然后输入密码和端口，可直接回车使用默认的
